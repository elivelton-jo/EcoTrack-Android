# 🌿 EcoTrack - Monitor de Consumo Sustentável

Projeto Android nativo desenvolvido para a disciplina de Desenvolvimento Mobile, focado em sustentabilidade e gestão de recursos (Água e Energia).

## 🚀 Funcionalidades Principais
- **Dashboard de Impacto:** Resumo automático no topo da tela calculando o impacto total de CO2 e economia de água.
- **Gestão de Consumo:** Cadastro detalhado de contas de luz e água com persistência em banco de dados local.
- **UX Avançada:** - Exclusão de registros através do gesto *Swipe* (deslizar para o lado).
    - Seleção de data via Calendário Nativo (*DatePicker*).
    - Tratamento de "Estado Vazio" quando não há registros.
- **Privacidade:** Fluxo de aceite de termos em conformidade com a **LGPD**.

## 🛠️ Arquitetura e Conceitos Aplicados
- **Linguagem:** Java.
- **Banco de Dados:** SQLite (CRUD completo).
- **POO (Orientação a Objetos):** Uso intensivo de Classes Abstratas, Herança e Polimorfismo.
- **Interface:** Material Design, RecyclerView, CardView e Floating Action Button.

## 📱 Telas do Sistema
1. **Tela Principal:** Lista de consumos e dashboard de resumo.
2. **Tela de Cadastro:** Formulário com validações e seletor de data.
3. **Tela LGPD:** Termos de privacidade para o primeiro acesso.
